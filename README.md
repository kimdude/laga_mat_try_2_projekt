# DT193G - Projekt
## Lagerhanterings-applikation 

Repot innehåller källkoden för en SPA-applikation. Applikationens syfte är att hantera produkter för det fiktiva företaget Laga Mat, som är en återförsäljare av köksmaskiner. 
Med hjälp av applikationen kan företaget hålla koll på och hanter lagersaldu, lägga till och ta bort produkter samt hanter användare. 

Webbapplikationen har skapats med Vue.js, Vue-router och Bootstrap för styling. Även JWT-decode används för att hämta info från token, som API:et använder för inloggning. 

Mer om API:et som används hittas i repot [https://github.com/kimdude/laga_mat_try_2_projekt.git](https://github.com/kimdude/laga_mat_try_2_projekt.git). Den färdiga webbapplikationen är [Laga Mat](https://lagamattake2.netlify.app/login).
